# Aplicativo tabela fipe
Aplicativo de consulta de tabela fipe utilizando a seguinte api pública: https://deividfortuna.github.io/fipe/

## Tecnologias utilizadas

- Kotlin
- Retrofit
- Crashlytics
- Firebase
- Padrões de projeto
